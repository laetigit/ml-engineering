**AI Weather Risk Assistant**



**Project Goal**



Create a small web interface where a user can:

* Enter a location
* View predictions and indicators for:

&nbsp;	🌡 Temperature forecast

&nbsp;	🌫 Air pollution level

&nbsp;	🌊 Sargassum algae risk

&nbsp;	🌋 Nearby seismic activity



The first version is mostly a data aggregation + simple prediction interface.

Later it can evolve into a real ML system and research-grade project.
